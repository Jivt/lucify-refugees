# migration

## Installation

### Dependencies

- Node + NPM
- Bower: `npm install -g bower`
- Gulp: `npm install -g gulp`
- Ruby + [RubyGems](https://rubygems.org/pages/download)
- Bundler: `gem install bundler`
- GDAL: `brew install gdal`
- topojson: `npm install -g topojson`

### Setup

1. `npm install`
2. `bower install`
3. `bundle install`
3. `gulp`
