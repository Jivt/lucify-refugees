# migration

## Installation

### Dependencies

- Node + NPM
- Ruby + [RubyGems](https://rubygems.org/pages/download)
- Bundler: `gem install bundler`
- GDAL: `brew install gdal`

### Setup

1. `npm install`
2. `./node_modules/bower/bin/bower install`
3. `bundle install`
3. `node ./node_modules/gulp/bin/gulp`
